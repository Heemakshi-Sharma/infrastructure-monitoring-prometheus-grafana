## Phase 3

- Install SNMP Exporter
- Monitor Cisco routers

- Interface monitoring
- Bandwidth monitoring

## Phase 4

- Configure Alertmanager
- Email alerts
- Slack notifications
- Dashboard improvements

## Phase 5

- Migrate the monitoring platform to GNS3 or EVE-NG
- Monitor enterprise network devices
- Build enterprise dashboards

# Long-Term Goal

Develop a production-style infrastructure monitoring solution capable of monitoring Linux servers, Docker containers, and Cisco network devices from a centralized Grafana dashboard.
