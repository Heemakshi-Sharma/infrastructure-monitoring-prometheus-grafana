## What is Grafana?

Grafana is an open-source visualization platform used to display metrics collected from monitoring systems such as Prometheus.

It helps administrators understand infrastructure performance through interactive dashboards.

# Why Grafana?

Grafana provides:

- Interactive dashboards
- Real-time visualization
- Multiple data source support
- Alerting capabilities
- Custom panels

# Dashboard

A dashboard is a collection of panels displaying different metrics.

Examples:

- CPU utilization
- Memory usage
- Disk usage
- Network traffic

# Data Source

A data source is the system from which Grafana retrieves monitoring data.

In this project:

Prometheus was configured as the data source.
