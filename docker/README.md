# Docker

Docker was used to deploy the monitoring stack inside the Ubuntu virtual machine.

Containers deployed:

- Prometheus
- Grafana

Docker simplified installation and allowed the services to run independently without manual package installation.

## Skills Learned

- Docker installation
- Pulling images
- Running containers
- Viewing containers
- Managing container lifecycle

## Commands Used

```bash
docker pull prom/prometheus

docker pull grafana/grafana

docker ps

docker images
```
